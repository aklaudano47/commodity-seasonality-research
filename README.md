# Commodity Seasonality Research

This repository contains an audited senior thesis research project evaluating whether seasonal return patterns in physical commodity markets persist under recursive out-of-sample testing.

## Summary

The project uses a recursive expanding-window walk-forward framework in R to evaluate seasonal behavior across commodity futures markets under transaction-cost assumptions.

The audited version clarifies:

* transaction-cost implementation;
* robustness testing;
* statistical interpretation;
* complete-calendar-year filtering;
* and portfolio-risk analysis.

## Key Result

The strongest illustrative equal-weight composite (Lumber, Wheat, and Corn) produced:

* 7.33% annualized return
* 0.73 Sharpe ratio
* -16.30% maximum drawdown

from 1978–2023 under the audited primary specification.
